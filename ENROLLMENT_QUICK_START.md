# 🎯 Quick Start: Student Enrollment

## 3 Ways to Enroll Students

### 1️⃣ Enroll Entire Department (Fastest!)
```
1. Go to: Semester → Course → Click "Enroll Students"
2. Find the blue box at top
3. Select department from dropdown
4. Click "Enroll Level"
✅ Done! All students from that department enrolled
```

### 2️⃣ Enroll Selected Students
```
1. Go to: Semester → Course → Click "Enroll Students"
2. Click on students in "Available Students" (left side)
3. Click "Enroll Selected" button
✅ Done! Selected students enrolled
```

### 3️⃣ Search & Enroll
```
1. Go to: Semester → Course → Click "Enroll Students"
2. Type name in search box
3. Click on the student
4. Click "Enroll Selected"
✅ Done! Student enrolled
```

## 📍 Where to Find It

**Path**: Admin Sessions → Pick a Session → Pick a Semester → See Courses

Each course has an **"Enroll Students"** button → Click it!

## 🆘 Common Questions

**Q: Will this generate exam tickets?**
A: No, enrolling students in a course doesn't generate tickets. Tickets are generated when you **activate an exam**. But students MUST be enrolled to receive tickets!

**Q: Can I undo enrollments?**
A: Yes! Click the trash icon next to any enrolled student to remove them.

**Q: What if I don't see a student?**
A: Use the search box, or check if they're already enrolled (check the right column).

**Q: Can Level Admins use this?**
A: Yes! Level Admins can enroll students from their assigned department only.

---

## 🎯 Remember

**Enrollment ≠ Ticket Generation**

- **Enroll students** = Allow them to take the course
- **Activate exam** = Generate tickets for enrolled students

**The flow**:
1. Enroll students in course ✅ (This page)
2. Instructor creates exam ✅
3. Admin activates exam ✅ (Tickets auto-generated for enrolled students)
4. Invigilator sees students with tickets ✅
